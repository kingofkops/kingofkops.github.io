  1. 注册Github账号

Github是全球最大的开源代码托管平台，拥有数百万用户。在本实验中，我们需要先注册一个Github账号(https://github.com/kingofkops)。

  1. 安装Git

Git是一款分布式版本控制系统，也是开源软件中使用最广泛的版本控制软件之一。在本实验中，我们需要使用Git来将博客部署到Github上，因此需要先安装Git。

在终端中 运行：

brew install git

  1. 安装Node.js

Node.js是一款基于Chrome V8引擎的JavaScript运行时，可以在服务器端运行JavaScript代码。在本实验中，我们需要使用Node.js来安装Hexo。

在终端中 运行：

brew install node

  1. 安装Hexo

Hexo是一款静态博客框架，可以将Markdown文档转换成静态网页。在本实验中，我们需要使用Hexo来搭建个人博客。

在命令行中输入以下命令，安装Hexo：

npm install -g hexo-cli

在命令行中输入以下命令，初始化Hexo：

hexo init blog

会在当前目录下创建一个名为blog的文件夹，用于存放Hexo博客的相关文件。

在命令行中输入以下命令，启动Hexo：

hexo server

启动Hexo本地服务器，并且可以在浏览器中通过http://localhost:4000访问。

  1. 选择主题

Hexo支持多种主题，我们可以根据自己的喜好选择一个主题，并进行相应的配置。

在Hexo博客所在的文件夹下，找到themes文件夹，可以看到默认的主题是landscape。我们可以在Github上搜索Hexo主题，并选择一个喜欢的主题下载。

git clone https://github.com/wd/hexo-fabric.git
